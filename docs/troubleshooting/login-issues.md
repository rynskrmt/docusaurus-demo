# ログインできない場合

ログインできない場合に考えられる原因と対処法をまとめました。

## パスワードを忘れた場合

1. ログイン画面の「パスワードをお忘れですか？」をクリックしてください。
2. 登録メールアドレスを入力し、「リセットメールを送信」を押します。
3. メールに記載されたURLから新しいパスワードを設定してください。

:::note
リセットメールが届かない場合は、迷惑メールフォルダをご確認ください。
:::

## 「アカウントが無効」と表示される場合

管理者がお客様のアカウントを一時停止、または無効化している可能性があります。
組織の管理者に直接お問い合わせください。

## 二段階認証コードが届かない場合

SMS認証の場合、電波状況の良い場所で再度お試しください。
認証アプリをご利用の場合は、アプリ内の時刻同期設定を確認してください。
