# 周知事項の閲覧

本アプリケーションのメイン機能である、会社からの重要なお知らせ（周知事項）の閲覧方法について解説します。

## 周知事項とは

会社全体、または所属チームに向けた重要な連絡事項が表示されます。
例：
- 規定の変更
- システムメンテナンスのお知らせ
- 提出期限の通知

## 閲覧手順

1. ログイン後のトップページに、最新の周知事項一覧が表示されます。
2. タイトルをクリックすると、詳細内容を確認できます。
3. 未読の周知事項には「未読」マークが付いています。必ず内容を確認してください。

## 過去の周知を検索する

過去の周知事項は、画面上部の検索バーからキーワードで検索可能です。
「あの件どうだったっけ？」と思った時は、まず検索してみましょう。
