# ユーザー管理

管理者は、組織内のユーザーを追加、編集、削除（無効化）することができます。

## ユーザーの招待

1. 管理画面のサイドメニューから「メンバー管理」を選択します。
2. 右上の「+ メンバーを招待」ボタンをクリックします。
3. 招待したいメンバーのメールアドレスを入力し、権限（管理者/一般ユーザー）を選択します。
4. 「送信」をクリックすると、招待メールが送信されます。

## ユーザー情報の編集

登録済みのユーザー情報は、以下の手順で変更できます。

1. メンバー一覧から対象のユーザーをクリックします。
2. 「編集」ボタンをクリックします。
3. 部署、役職、権限などを変更し、「保存」をクリックします。

## ユーザーの無効化

退職などでアカウントを利用停止にする場合は、アカウントを削除せず「無効化」することをお勧めします。
無効化することで、過去の投稿履歴などを保持したままログインを停止できます。
