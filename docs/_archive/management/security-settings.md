# セキュリティ設定

組織全体のセキュリティポリシーを設定します。

## パスワードポリシー設定

ユーザーに強力なパスワード設定を強制することができます。

- **最小文字数**: 8文字以上、10文字以上など
- **文字種の強制**: 英数字記号の混在必須など
- **有効期限**: パスワードの定期変更を求める場合に設定

## 二段階認証（2FA）の強制

管理画面から、全ユーザーに対して二段階認証の利用を必須化できます。
セキュリティ強化のため、有効にすることを強く推奨します。

1. 管理画面 > セキュリティ設定 へ移動
2. 「二段階認証を必須にする」をオンにする
3. 設定を保存

## IPアドレス制限

特定のIPアドレス（社内ネットワークなど）からのアクセスのみを許可する設定です。
リモートワークなどで社外からのアクセスが必要な場合は、VPN経由での接続を検討してください。
