# プロフィールの設定

## 基本情報の入力

自分のプロフィールを充実させることで、他のメンバーとのコミュニケーションが円滑になります。

1. 右上のアイコンをクリックし、「プロフィール」を選択します。
2. 「編集」ボタンをクリックします。
3. 部署、役職、自己紹介などを入力します。
4. プロフィール画像を設定すると、発言者が分かりやすくなります。

## 通知設定

必要な通知だけを受け取るように設定をカスタマイズしましょう。

- **メール通知**: 重要な通知（メンションなど）のみオンにするのがおすすめです。
- **プッシュ通知**: ブラウザのプッシュ通知を許可すると、リアルタイムで反応できます。
