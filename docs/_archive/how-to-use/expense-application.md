# 経費精算の申請

経費が発生した場合は、月末までに必ず申請を行ってください。

## 申請が必要なもの

- **交通費**: 業務上の移動にかかった費用
- **会議費**: 打ち合わせ時の飲食費など（5,000円/人 以下）
- **交際費**: 接待など（事前申請必須）
- **消耗品費**: 業務に必要な物品購入

## 申請フロー

1. 「ワークフロー」メニューから「経費申請」を選択。
2. 領収書の画像を添付（必須）。
3. 金額、日付、用途を入力。
4. 「申請する」をクリック。
5. 上長の承認後、翌月の給与とともに振り込まれます。

:::tip ポイント
領収書の宛名は必ず「株式会社〇〇」としてもらってください。
:::
