# エラーコード一覧

システム利用中に表示される主なエラーコードとその意味です。

| エラーコード | 意味                   | 対処法                                                              |
| ------------ | ---------------------- | ------------------------------------------------------------------- |
| **E-001**    | ネットワーク接続エラー | 通信環境を確認し、再読み込みしてください。                          |
| **E-103**    | 権限不足               | この操作を行う権限がありません。管理者にご確認ください。            |
| **E-404**    | ページが見つかりません | 正しいURLか確認してください。すでに削除されている可能性があります。 |
| **E-500**    | サーバーエラー         | システム側の障害です。しばらく時間をおいてアクセスしてください。    |

上記以外のエラーが表示された場合は、エラーコードと発生状況を添えてサポートデスクまでご連絡ください。
