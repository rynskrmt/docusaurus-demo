# 最初のステップ

サービスを利用開始するにあたっての基本的な手順をご案内します。

## 1. アカウントの初期設定

招待メールが届いたら、以下の手順でアカウントを設定してください。

1. メールのリンクをクリックして、パスワード設定画面へ移動します。
2. 強力なパスワードを設定してください（英数字記号を含む8文字以上推奨）。
3. プロフィール画像と基本情報を登録します。

## 2. チームへの参加

所属する部署やプロジェクトのチームに参加しましょう。

- 左サイドバーの「チーム一覧」をクリック。
- 参加したいチームの「参加リクエスト」ボタンを押します。
- 管理者の承認後、チーム内の情報を閲覧できるようになります。

## 3. まずは検索してみましょう

画面上部の検索バーから、興味のあるキーワードで検索してみてください。
先輩社員が残した多くのナレッジが見つかるはずです。
